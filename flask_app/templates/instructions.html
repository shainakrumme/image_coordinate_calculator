Instructions

The Image Coordinate Calculator will calculate evenly spaced (x, y) coordinate pairs given the dimensions and corner points of an image.



Input


Image Dimensions
  • The image dimensions are a tuple defined as (width, height).
  • The width and height values are positive integers and represent pixel measurements.
  • The width and height can also be thought of as the number of columns and the number of rows, respectively.

Corner Points
  • The corner points may be provided in any order.
  • The corner points are a list of four tuples or (x, y) coordinate pairs.
  • The x and y values are positive numbers (either integers or decimals).
  • Each tuple represents the location of a pixel in the image.


Output

  • The results are rounded to two decimal places.
  • The results are given as both an array and a plot.
